Анализ:
Так как числа вводятся с клавиатуры, то пользователь нолжен вводить
эти числа. Введём переменную (bool found1 = false), которая будет
отвечать за продолжительность последовательности. Введём переменную
(bool found2 = true;), которая будет отвечать за определение
упорядоченности последовательности по возрастанию, т.е. определять
упорядочины ли числа в последовательности по возрастанию или нет.
Вводим первое число последовательности b. После этого провяряем
равняется ли число b 0 (b=0), если да, то found1 = true. Остальные
числа последовательности будем вводить в цикле while (found1 == false).
В цикле пользователь будет вводить число a, после чего число будет
сравниватся с 0 (if (a == 0)), если a=0, то found1 = true ( это
означает последнее выполнение цикла). Потом всё ещё в этом цикле while
сравниваем предыдущее число с введённым сейчас , если a < b и a!=0, то
found2 = false. После присваиваем b a (b = a). После выхода из цикла
проверяем переменную found2, если found2 == true, то выводим "Числа в 
последовательности упорядочены по возрастанию", иначе выводим "Числа в 
последовательности не упорядочены по возрастанию".
