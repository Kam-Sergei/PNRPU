Анализ:
Так как n не указано, то пользователь должен ввести количество элементов в 
последовательности (n). Так как последователность начинается с 1, то
введём переменную int i=1. Введём переменную float max=0, в которую будет
присваиваться максимальное значение последовательности. Используем цикл
while (i<=n), в котором будет вычислятся элемент последовательности
a = sin(n + i / n) , после чего будет проверяться является ли элемент 
последовательности первым (if (i == 1)), если является, то присваиваем
max = a и k = 1, где k-количество элементов, совпадающих с максимальным,
иначе сравниваем значение этого элемента с максимальным (else if (a > max)),
если условие выполняется, то присваиваем max = a и k = 1, иначе сравниваем 
значение этого элемента с максимальным (else if (a == max)), если условие 
выполняется, то прибавляем к k 1 (k = k+1). После проверок переходим к 
следующему элементу последовательности, для этого прибавляем к i 1 (i++).
После выполнения цикла ,так как надо найти максимальный элемент в 
последовательности (его значение и номер), выведем номер максимального
значения в последовательности (j) и максимальное значение в последовательности
(max).
